# 🧘 Zen Image

> **Simple. Pure. Zen.**  
> 복잡함은 덜어내고 본질에 집중합니다. 가장 간결하고 강력한 심플한 이미지 유틸리티.

![App Icon](src/assets/app-icon.png)

## ✨ 주요 기능

Zen Image는 이미지 작업을 위한 4가지 핵심 도구를 제공합니다.

1. **📏 Image Resizer**
   - 품질 손실 없이 순식간에 이미지 크기를 변경합니다.
   - 무압축 고품질 리사이징을 지원합니다.

2. **✂️ Image Cropper**
   - 다양한 프리셋 비율로 이미지를 정밀하게 자릅니다.
   - 실시간 미리보기를 통해 최적의 구도를 잡을 수 있습니다.

3. **🖼️ Auto Image Trimmer**
   - 이미지 배경을 자동으로 분석하여 객체만 정교하게 남겨줍니다.
   - 투명 배경 이미지(누끼 따기) 제작에 최적화되어 있습니다.

4. **🔄 Image Converter**
   - 이미지를 PNG, JPG, WEBP 등 원하는 포맷으로 자유롭게 변경합니다.
   - 품질 조절 옵션을 통해 용량과 화질의 균형을 맞출 수 있습니다.

## 🛠️ 기술 스택

이 프로젝트는 별도의 프레임워크 없이 순수 웹 표준 기술을 사용하여 구축되었습니다.

- **Language**: Vanilla JavaScript, HTML5, CSS3
- **Styling**: Vanilla CSS, Tailwind CSS (Design System)
- **Architecture**: Web Components (Custom Elements)
- **Core API**: HTML5 Canvas API (Image Processing)

## 🚀 시작하기

별도의 빌드 과정 없이 정적 파일 서버를 통해 바로 실행할 수 있습니다.
